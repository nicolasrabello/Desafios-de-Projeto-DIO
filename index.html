Client X Server

(voltar)

A comunicação na internet é baseada em um modelo chamado de client / server. Sendo que o cliente está consumindo os dados na rede. Na outra ponta está o servidor, que é o dispositivo que hospeda esses dados.

Cliente-servidor é baseada em troca de mensagens, onde o cliente envia requisições e o servidor responde com as informações requisitadas.

O que é HTML

(voltar)

HTML é a sigla para HyperText Markup Language (Linguagem de Marcação de Hipertexto, em inglês) com o desenvolvimento dos navegadores o HTML passou a ser bastante utilizados nas criações de sites, que é basicamente uma linguagem de marcação, onde os elementos de uma página são definidos por marcadores conhecido como "Tags" que pode ser formatados como parágrafos, sublinhados e links, até conteúdos multimídia, como imagens e vídeos.

Exemplos de Tags

(voltar)

"p" Tag de parágrafo "/p"

Essa tag formata qualquer texto entre a tag de abertura “p” e a tag de fechamento “/p” como um parágrafo padrão

"strong" Tag de negrito "/strong"

Esta tag irá formatar qualquer texto entre as tags como negrito

"i" Tag em itálico "/i"

Esta tag irá formatar qualquer texto entre a tag de abertura e a tag de fechamento como itálico.

"u" Tag de sublinhados "/u"

Esta tag formata todo texto como sublinhado
